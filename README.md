# **MyFlix App Client**

This second part of the My Flix App is the front-side of the application. It allows users to access the application and interact with it. Users can register a new account, browse a list of movies with information about the movie, such as its description, information about its director or the genre and add the movie to the user favorite movies list. Users can also update their personal details and delete their account.

## **Features**

* Register an account
* Get a list of all the movies
* Search for an specific movie
* Get information about a movie (Description, Image)
* Get information about a Director 
* Get information about a Genre
* Add a movie to favorites
* Update User details
* Remove a movie from favorites
* Delete an account
